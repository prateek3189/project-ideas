  • Rs. 199.00 debited from A/c **XXXX1234** - Bank account debits
  • via UPI to MERCHANT - UPI transactions
  • Txn ID 1234567890 - Transaction reference
  • UTR 2XXXXXXXXX - UPI transaction reference
