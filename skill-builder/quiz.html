Question 1 Single Choice

What is the best semantic element to use for the main content of a webpage?

Select the correct answer:

Your answer: B) <main>

Correct answer: B) <main>

The <main> element represents the main content of the document. It should contain the primary content that is directly related to or expands upon the central topic of the document. This is the most semantic choice for the main content area.

Question 2 Multiple Choice

Which of the following are inline elements in HTML?

Select all correct answers:

Your answers: A) <span>, B) <em>, D) <strong>

Correct answers: A, B, and D

<span>, <em>, and <strong> are all inline elements that flow within the text. <article> is a block-level element used for self-contained content.

Question 3 True/False

The <nav> element is used specifically for site navigation.

Select true or false:

Your answer: True

Correct answer: True

The <nav> element is specifically designed for navigation purposes. It represents a section of a page that contains navigation links, such as menus, tables of contents, or indexes.

Question 4 Single Choice

Which TypeScript type would you use for a variable that can be either a string or null?

Select the correct answer:

Your answer: B) string | null

Correct answer: B) string | null

The union type operator | allows a variable to accept multiple types. string | null means the variable can be either a string or null, which is perfect for optional string values.

Question 5 Single Choice

What is the purpose of a type guard in TypeScript?

Select the correct answer:

Your answer: B) To narrow down union types at compile time

Correct answer: B) To narrow down union types at compile time

Type guards help TypeScript understand which specific type a union type variable holds at a given point in the code. This allows for safe property access and method calls on the narrowed type.
