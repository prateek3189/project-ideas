What are Union Types?

Union types allow a variable to hold values of multiple types. They're perfect for scenarios where a value could be one of several types.

Key Points:

  • Use the | operator to combine types
  • TypeScript will enforce that values match one of the union types
  • Common use case: handling different data formats
Union Type: A type that can be one of several types

Union types are a fundamental TypeScript feature that allows variables, parameters, and return types to accept multiple types. They're written using the pipe operator (|).

Example: string | number means a value can be either a string or a number.

Real-World Example

Let's see how union types work in practice with a common scenario: handling user input that could be a string or number.

Type Definition

type UserInput = string | number;

function processInput(input: UserInput): string {
    if (typeof input === 'string') {
        return input.toUpperCase();
    } else {
        return input.toString();
    }
}

What's happening:

  • We define UserInput as a union of string and number
  • The function accepts either type safely
  • Type guards (typeof) help narrow the type
