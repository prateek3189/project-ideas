Tag Guidelines

Do's

  • • Use lowercase, hyphenated names
  • • Be specific and descriptive
  • • Include examples in definitions
  • • Merge duplicate tags regularly
  • • Archive unused tags quarterly

Don'ts

  • • Don't create overly broad tags
  • • Avoid abbreviations without context
  • • Don't create tags for one-off issues
  • • Avoid duplicate or similar tags
