Content Security

Security Notice: Forbid inline scripts in MFEs; sandbox iframes for enhanced security.

Content Security Policy (CSP)

All widgets must comply with strict CSP rules. Inline scripts are prohibited, and all external resources must be whitelisted. Use nonces or hashes for legitimate inline content.

Widget Sandboxing

Widgets run in isolated iframe sandboxes with restricted permissions. This prevents malicious widgets from accessing sensitive data or performing unauthorized actions.

Data Isolation

Each widget operates in its own context with controlled data access. Cross-widget communication is mediated through secure event channels with validation and sanitization.
